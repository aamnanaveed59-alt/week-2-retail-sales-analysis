# week-2-retail-sales-analysis
# 📊 Retail Sales Analysis Dashboard | LogicStack Data Analysis Internship (Week 2)

## 📌 Project Overview

This project was completed as part of the **LogicStack Data Analysis Internship (July 2026)**.

The objective was to analyze a retail sales dataset using **Microsoft Excel** and **Power BI**, perform KPI calculations, generate business insights, and build an interactive dashboard for business decision-making.

---

# 🛠️ Tools Used

* Microsoft Excel
* Power BI

---

# 📋 Project Tasks

## ✅ Task 1: Excel Analysis

Created Pivot Tables for:

* Total Sales by Product Category
* Total Sales by Gender
* Total Quantity by Product Category
* Average Age by Product Category

---

## ✅ Task 2: KPI Calculations

Calculated the following Key Performance Indicators (KPIs):

* Total Sales
* Total Transactions
* Average Sales
* Total Quantity Sold
* Highest Transaction Value
* Lowest Transaction Value

---

## ✅ Task 3: Business Insights

Generated business insights based on the analysis, including:

* Best-performing product category
* Gender-wise sales comparison
* Most active customer age group
* Sales performance observations
* Quantity sold analysis

---

# 📊 Power BI Dashboard

The dashboard includes:

### KPI Cards

* Total Sales
* Total Transactions
* Total Quantity Sold
* Average Sales

### Visualizations

* Bar Chart – Sales by Product Category
* Pie Chart – Sales by Gender
* Column Chart – Quantity Sold by Product Category
* Line Chart – Sales Trend by Date
